# CRWN-Clothing

CRWN Clothing is a modern and user-friendly e-commerce platform that allows users to browse, purchase and track their favorite clothing items online. The application has been developed using React, which is a popular and efficient frontend library, and Firestore, a robust NoSQL document database from Firebase, as the backend.

​

One of the key features of the application is its ability to provide a personalized user experience. CRWN Clothing makes use of user authentication and authorization features offered by Firestore to allow users to sign up and log in securely. This enables customers to create and maintain their profile, save their favorite items, and track their order history.

​

The application has been designed to provide an intuitive and responsive user interface with a seamless shopping experience. It offers a wide range of clothing options, including shirts, pants, dresses, and accessories, among others. Customers can use various filters and sorting options to search for specific items and add them to their cart with a single click.

​

CRWN Clothing also makes use of Redux to manage the application state efficiently. It allows for better organization and management of data across the application, improving performance and responsiveness. Typescript has been used to add static typing to the application, making it easier to maintain and debug.

​

In addition to the above-mentioned technologies, CRWN Clothing also utilizes Scss, a powerful CSS preprocessor, to enable efficient styling and theming of the application. The application also makes use of GraphQl to retrieve and display data from the Firestore database more efficiently, improving the overall performance of the application.
